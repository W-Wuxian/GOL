# GOL repository
A CPP implementation of the Game Of Life (GOL) problem with R as a visualizer.

![](./Random/Random.gif)

# running cmd :
In a shell use the following command:
```sh
User@My-Linux-Desktop:~$ cd ~/GOL/ && make help
```
# Dependencies
- Make
- g++ as c++ compiler (std17)
- R used to plot the GOL state
